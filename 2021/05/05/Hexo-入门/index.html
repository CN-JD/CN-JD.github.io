Hexo 入门

发表于 2021-05-05 分类于 前端 ， Node.js

简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

无需服务器，可以利用 github、gitee（码云）等托管在互联网中，实现网上部署自己的博客

入门篇

环境准备

  • Node.js
  • Git

安装

测试

验证 hexo 版本

搭建第一个博客

  1. 初始化 hexo init <folder> ，名称任取

  2. 进入创建好的目录

  3. 去除静态网站文件

  4. 生成静态网站文件

  5. 部署在本地环境

  6. 访问网站 [localhost](http://localhost:4000/) （ CTRL + C 停止项目）

初始化后的目录结构

命令 hexo g 会 生成 public 目录，其用于存放静态网页文件资源

设置

修改 _config.yml 基本配置（部分）

网站

参数 描述
title 网站标题
subtitle 网站副标题
description 网站描述
keywords 网站的关键词
author 您的名字
language zh-Hans 或 zh-CN
timezone Asia/Shanghai

重新部署：

  • hexo clean
  • hexo g
  • hexo s
