BufferedReader

发表于 2021-05-22 分类于 后端 ， Java

缓冲字符读入器

缓冲字符读入器（BufferedReader）用于从基于字符的输入流中读取文本

它可以用于通过 readLine() 方法逐行读取数据。 它使性能快速提高。 它继承了Reader类。

Java.io.BufferedReader class:

类构造方法

构造方法 描述
BufferedReader(Reader rd) 它用于创建一个缓冲字符输入流，该流使用输入缓冲区的默认大小
BufferedReader(Reader rd, int size) 它用于创建使用输入缓冲区指定大小的缓冲字符输入流

类方法

方法 描述
int read() 它用于读取单个字符
int read(char[] cbuf, int off, int len) 它用于读取字符数组的一部分
boolean markSupported() 它用于测试标记和重置方法的输入流支持。
String readLine() 它用于读取一行文本
boolean ready() 它用于测试输入流是否准备好被读取
long skip(long n) 它用于跳过字符
void reset() 它将该流重新定位在该输入流上最后一次调用 mark() 方法位置
void mark(int readAheadLimit) 它用于标记流中的当前位置
void close() 它关闭输入流并释放与该流相关联的任何系统资源
