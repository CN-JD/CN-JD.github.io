字符读出器

字符读出器（Reader）是一个用于读取字符流的抽象类。子类必须实现的唯一方法是 read(char[]，int，int) 和 Close()。大多数子类将覆盖某些方法，以提供更高的效率、额外的功能

一些实现类包括 BufferedReader、CharArrayReader、FilterReader、InputStreamReader、PipedReader、StringReader

属性

修饰符和类型 字段 描述
protected Object lock 用于同步此流上的操作的对象

类构造方法

修饰符 构造方法 描述
protected Reader() 它创建了一个新的字符流阅读器，其关键部分将在阅读器本身上同步
protected Reader(Object lock) 它创建一个新的字符流读取器，其关键部分将在给定对象上同步。

类方法

修饰符和类型 方法 描述
abstract void close() 它关闭流并释放与它相关的任何系统资源
void mark(int readAheadLimit) 它标记了流中的当前位置
boolean markSupported() 流是否支持 make() 操作
int read() 它读取单个字符
int read(char[] cbuf) 将字符读入一个数组
abstract int read(char[] cbuf, int off, int len) 将字符读入一个数组的一部分
int read(CharBuffer target) 它试图将字符读入指定的字符缓冲区
boolean ready() 流是否准备好被读取
void reset() 重置数据流
long skip(long n) 跳过字符
