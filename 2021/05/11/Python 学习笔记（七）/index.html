类

创建和使用类

根据类来创建对象被称为 实例化

  • 方法 __init__() ：每当实例化一个对象时就会运行。（类似 Java 构造方法）
  • 形参 self : 指向实例本身，用于访问类中的属性和方法。（类型 Java 的 this ）
  • 类的属性在 __init__() 方法中定义，且必须有初始值（形参赋值或有默认值）
  • 可以通过 对象.属性 访问或修改值，或使用方法包装后修改
  • class 首字母不大写

person.py

继承

通过 class 子类名(父类名) 表示继承关系

  • super() 相当于父类，将父类与子类关联

  • 子类在初始化时必须调用父类的初始化方法 super().__init__(..形参)

  • 重写方法：子类方法名与父类方法名相同，子类会覆盖父类方法

student.py

导入类

将类写在其他 .py 文件中称为模块，程序使用时引入，文件名即模块名

  • 导入单个类 from 模块名 import 类名

  • 导入多个类 from 模块名 import 类名, 类名

  • 导入所有类 from 模块名 import * 【不推荐使用，类名可能冲突】

  • 导入整个模块 import 模块名 , 通过 模块名.类名 调用

类编写风格

  • 驼峰式命名法，每个单词首字母大写，不使用下划线
  • 实例名都小写，单词间用下划线分隔
  • 类后面要加个文档字符串描述功能
  • 类中用一个空行分隔方法，模块中用两个空行分隔类
  • 向 import 标准库模块在一个空行 import 你编写的模块
