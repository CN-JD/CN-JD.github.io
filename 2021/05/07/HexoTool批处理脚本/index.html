HexoTool批处理脚本

发表于 2021-05-07 分类于 脚本 ， Bat

简介

批处理定义：批处理文件是将一系列命令按一定的顺序集合为一个可执行的文本文件，其扩展名为BAT 或者 CMD 。这些命令统称批处理命令。

其实，我们在 CMD 中输入的例如 cd、ping 命令都是批处理命令。

在 windows 平台下编写一个批处理文件，有利与我们处理一些繁琐且重复的活动，例如本文介绍的利用批处理命令编写一个用于管理 Hexo 博客的批处理脚本。

快速上手

那么，如何编写一个 Bat 批处理文件呢，其实很简单。

  • 新建一个空白 TXT 文本，复制以下内容并保存退出

  • 将创建的文本文件的后缀 .txt 改为 .bat

  • 双击执行

常用批处理命令

以下介绍的批处理命令都是我在编写该脚本时所涉及到的，更多命令可以在互联网中查找

注释

:: 或者 rem

chcp

显示或设置活动代码页编号

解决汉字显示的乱码问题，65001 表示使用 UTF-8 编码格式，放在脚本文件开头

@echo off

echo off 关闭后面命令回显

@ 关闭当前命令回显

不显示执行命令的语句，精简内容显示

title

设置命令提示窗口的窗口标题

echo

显示消息

在消息末尾一定有一个空格，不然可能会报错

echo.

换行

set

显示、设置或删除 cmd.exe 环境变量

设置变量并静态赋值

set /p

设置变量，显示提示信息，等待用户输入并赋值

去除变量两边空格

使用变量

%变量名%

if

执行批处理程序中的条件处理

多重判断格式

exist

判断文件是否存在

not

条件取反

&

依次执行命令，不管前面命令执行是否成功，后面命令依次执行

&&

依次执行命令，前面执行成功才会执行后面的命令

：

声明一个标签

:~

截取变量值部分字段

goto

定向到批处理程序中带标签的行

跳转到指定的标签

cd

跳转目录

pause

暂停批处理程序，并显示以下消息:
请按任意键继续. . .

让消息停留

exit

退出当前批处理脚本

cls

清除屏幕

start

启动一个单独的窗口以运行指定的程序或命令

用默认浏览器打开一个网址

dir /b

只显示目录文件

del

删除一个或多个文件
