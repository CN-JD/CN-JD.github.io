DataOutputStream

发表于 2021-05-21 分类于 后端 ， Java

数据输出流

数据输出流（DataOutputStream）允许应用程序以独立于机器的方式向输出流写入原始的 Java 数据类型，这些数据以后可以被数据输入流读取

类声明

java.io.DataOutputStream class:

类构造方法

构造方法 描述
DataOutputStream(OutputStream out) 创建一个新的数据输出流，以将数据写入指定的底层输出流

类方法

方法 描述
int size() 它用于返回写入数据输出流的字节数
void write(int b) 它用于将指定字节写入底层输出流
void write(byte[] b, int off, int len) 它用于将 len 字节的数据写入输出流
void writeBoolean(boolean v) 它用于将布尔值以 1 字节的值写入输出流。
void writeChar(int v) 它用于将字符写到输出流中，作为一个 2 字节的值
void writeChars(String s) 它用于将字符串作为一个字符序列写入输出流
void writeByte(int v) 用于将一个字节作为 1 字节的值写入输出流
void writeBytes(String s) 用于将字符串作为一个字节序列写入输出流
void writeInt(int v) 用于向输出流写入一个整型
void writeShort(int v) 用于向输出流写入一个短整形
void writeLong(long v) 用于向输出流写入一个长整形
void writeUTF(String str) 用于以可移植的方式使用 UTF-8 编码向输出流写入一个字符串
void flush() 用于刷新数据输出流
