文件权限

文件权限类（FilePermission）包含与目录或文件相关的权限

所有权限都与路径相关。 路径可以有两种类型：

  1. *D:\\IO\\*- ：它表示权限递归地与所有子目录和文件关联
  2. D:\\IO\\* : 它表明该权限与该目录中的所有目录和文件（子目录除外）相关联
