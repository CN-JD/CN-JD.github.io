发表于 2021-05-21 分类于 后端 ， Java

控制台

控制台类（Console）用于从控制台获取输入。它提供了读取文本和密码的方法

如果您使用 Console 类读取密码，它将不会显示输入内容给用户

java.io.Console 在内部附加了系统控制台。Console 类是从1.5开始引入的

System 类提供了一个静态方法 console()，它返回 console 类的单例实例

类方法

方法 描述
Reader reader() 它用于检索与控制台关联的读取器对象
String readLine() 它用于从控制台读取一行文本
String readLine(String fmt, Object… args) 它提供格式化的提示符，然后从控制台读取一行文本
char[] readPassword() 用于读取未在控制台中显示的密码
char[] readPassword(String fmt, Object… args) 它提供一个格式化的提示，然后读取没有显示在控制台的密码
Console format(String fmt, Object… args) 用于向控制台输出流写一个格式化的字符串
Console printf(String format, Object… args) 它用于向控制台输出流写入一个字符串
PrintWriter writer() 它用于检索与控制台相关的 PrintWriter 对象
void flush() 它用于刷新控制台
