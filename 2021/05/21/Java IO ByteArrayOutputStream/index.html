ByteArrayOutputStream

发表于 2021-05-21 分类于 后端 ， Java

字节数组输出流

字节数组输出流（ByteArrayOutputStream）用于将公共数据写入多个文件，在此流中，数据被写入字节数组，此字节数组以后可以写入多个流。

ByteArrayOutputStream 保存数据的副本，并将其转发到多个流

ByteArrayOutputStream 的缓冲区会根据数据自动增长

类构造方法

构造方法 描述
ByteArrayOutputStream() 创建初始容量为 32 字节的字节数组输出流，但其大小会在必要时增加
ByteArrayOutputStream(int size) 创建具有指定大小(以字节为单位)的缓冲区容量的字节数组输出流

类方法

方法 描述
int size() 它用于返回缓冲区的当前大小
byte[] toByteArray() 它用于创建一个新分配的字节数组
String toString() 它用于将内容转换为使用平台默认字符集的字符串解码字节
String toString(String charsetName) 用于将内容转换为使用指定的字符集名称解码的字符串
void write(int b) 它用于将指定的字节写入字节数组输出流
void write(byte[] b, int off, int len 用于从偏移量 off 开始，从指定的字节数中写入 len 字节到字节数输出流
void writeTo(OutputStream out) 用于将字节数组输出流的全部内容写到指定的输出流中
void reset() 将此字节数组输出流的 count 字段重置为零，以便丢弃输出流中当前累积的所有输出
void close() 它用于关闭字节数组输出流

ByteArrayOutputStream 类将通用数据写入两个文件 f1.txt 和 f2.txt
