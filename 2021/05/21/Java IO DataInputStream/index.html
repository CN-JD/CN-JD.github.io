发表于 2021-05-21 分类于 后端 ， Java

数据输入流

数据输入流（DataInputStream）允许应用程序以独立于机器的方式从输入流中读取原始数据

类声明

java.io.DataInputStream class:

类构造方法

构造方法 描述
DataInputStream(InputStream in) 创建使用指定的底层 InputStream 的 DataInputStream

类方法

方法 描述
int read(byte[] b) 它用于从输入流中读取字节数
int read(byte[] b, int off, int len) 它用于从输入流中读取 len 字节的数据
int readInt() 用于读取输入字节并返回一个 int 值
byte readByte() 用于读取并返回一个输入字节
char readChar() 它用于读取两个输入字节并返回一个字符值
double readDouble() 它用于读取八个输入字节并返回一个双精度值。
boolean readBoolean() 用于读取一个输入字节，如果字节非零则返回 TRUE，如果字节为零则返回 FALSE
int skipBytes(int x) 它用于跳过输入流中 x 字节的数据
String readUTF() 它用于读取使用 UTF-8 格式编码的字符串
void readFully(byte[] b) 它用于从输入流中读取字节并将其存储到缓冲区数组中
void readFully(byte[] b, int off, int len) 它用于从输入流读取 len 字节
