BufferedOutputStream

发表于 2021-05-21 分类于 后端 ， Java

缓冲输出流

缓冲输出流（BufferedOutputStream）用于缓冲一个输出流

它在内部使用缓冲区来存储数据，它比直接将数据写入流中的效率更高。因此，它可以提高性能。

构造方法 描述
BufferedOutputStream(OutputStream os) 它创建新的缓冲输出流，用于将数据写入指定的缓冲区输出流
BufferedOutputStream(OutputStream os, int size) 它创建新的缓冲输出流，用于将数据写入指定的缓冲区大小的输出流中

方法 描述

void flush() 它将刷新缓冲输出流
