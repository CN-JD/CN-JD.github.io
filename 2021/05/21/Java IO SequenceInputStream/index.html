SequenceInputStream

发表于 2021-05-21 分类于 后端 ， Java

序列输入流

序列输入流（SequenceInputStream）用于从多个流读取数据

它按顺序(一个接一个)将多个输入流合并成一个输入流

类声明

Java.io.SequenceInputStream class:

类构造方法

构造方法 描述
SequenceInputStream(InputStream s1, InputStream s2) 创建一个新的输入流，按顺序读取两个输入流的数据，首先是 s1，然后是 s2
SequenceInputStream(Enumeration e) 通过读取类型为 InputStream 的枚举数据创建一个新的输入流

类方法

方法 描述
int read() 它用于从输入流中读取下一个字节
int read(byte[] ary, int off, int len) 它从给定的偏移量开始，将指定长度的输入流中的字节读入到字节数组
int available() 它用于返回可从输入流读取的最大字节数
void close() 它用于关闭输入流
