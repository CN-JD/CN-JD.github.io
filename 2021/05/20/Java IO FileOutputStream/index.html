FileOutputStream

发表于 2021-05-20 分类于 后端 ， Java

文件输出流

文件输出流（FileOutputStream）是用于向文件写入数据的输出流

如果你必须把原始值写进一个文件，请使用 FileOutputStream 类

你可以通过 FileOutputStream 类写入面向字节和面向字符的数据。但是，对于面向字符的数据，最好使用 FileWriter 类

类方法

方法 描述
protected void finalize() 它用于清理与文件输出流的连接
void write(byte [] ary) 它用于将整个字节数组写入文件输出流
void write(byte [] ary，int off，int len) 它用于将从偏移量 off 开始的字节数组中 len 长度字节写到文件输出流中
void write(int b) 用于将指定的字节写入文件输出流
FileChannel getChannel() 它用于返回与文件输出流关联的文件通道对象
FileDescriptor getFD() 它用于返回与流关联的文件描述符对象
void close() 用于关闭文件输出流
