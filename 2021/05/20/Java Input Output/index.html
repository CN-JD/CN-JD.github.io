Java Input/Output

发表于 2021-05-20 分类于 后端 ， Java

本 Java I/O 系列参考来源于该 地址

Java I/O (Input 和 Output) 用于处理输入和产生输出

Java 使用流的概念来使 I/O 操作更快，java.io 包包含了输入和输出操作所需的所有类

我们可以通过 Java I/O API 在 Java 中进行文件处理

流（Stream）

流（Stream）是一个数据序列，在 Java 中，流是由字节组成的。它之所以被称为流，是因为它就像一条不断流动的水流

在 Java 中，为我们自动创建了3个流，所有这些流都与控制台相连

  1. System.out：标准输出流
  2. System.in: 标准输入流
  3. System.out: 标准错误流

向控制台打印输出和错误消息的代码

从控制台获取输入的代码

输入输出流

输出流（OutputStream）：Java 应用程序使用输出流将数据写入一个目的地，它可以是一个文件、一个数组、外围设备或套接字

输入流（InputStream）：Java应用程序使用输入流从源中读取数据，它可以是文件，阵列，外围设备或套接字

输出流类

OutputStream 类是一个抽象的类，它是所有代表字节输出流类的超类，输出流接受输出字节并将其发送到某个接收器

输出流的实用方法

方法 描述
public abstract void write(int b) throws IOException 将字节写入当前输出流
public void write(byte b[]) throws IOException 将字节数组写入当前输出流
public void flush() throws IOException 刷新当前输出流
public void close() throws IOException 关闭当前输出流

输入流类

InputStream 类是一个抽象类，它是所有代表字节输入流类的超类

输入流的实用方法

方法 描述
public abstract int read() throws IOException 从输入流中读取下一个字节的数据，在文件结束时返回 -1
public int available() throws IOException 返回可以从当前输入流中读取的字节数的估计值
public void close() throws IOException 关闭当前输入流
