文件

读取文件

  • Python 会在合适的时候自动关闭文件
  • 没有指定路径会在当前目录下查找
  • 一般在当前目录下创建文件夹存放文件，例 test_files\filename.txt
  • 使用 rstrip() 去除多余空白行

逐行读取

函数名 作用
open 打开文件
read 读取文件所有内容
readlines 逐行读取文件成一个列表

写入文件

  • 调用 open() 方法时输入第二个实参 w 以写入模式打开文件
  • r 默认只读模式
  • w 写入模式（无文件创建，有文件清空）
  • a 附加模式（无文件创建，有文件添加在末尾）
  • r+ 读取和写入模式
  • Python只能将字符串写入文本文件。要将数值数据存储到文本文件中，必须先使用函数 str()将其转换为字符串格式
