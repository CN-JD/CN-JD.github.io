数字计算

数值运算符

操作符 操作
+ 加
- 减

/ 浮点除

abs() 绝对值

内置函数

函数 描述
range(stop) 返回从 0 到 stop − 1 的整数列表
range(start, stop) 返回从 start 到 stop−1 的整数列表
range(start, stop, step) 返回从 start 到 stop 的整型列表，步长为 step
type(x) 返回 x 的 Python 数据类型
int(x) 返回 x 转换为整型的值。x 可以是数值或字符串
float(x) 返回 x 转换为浮点型的值。x 可以是数值或字符串
round(x) 返回 x 最近的整数值（作为浮点型）

Math 库函数

Python 数学 解释
pi $\pi$ $\pi$ 的近似值
e $e$ $e$ 的近似值
sqrt(x) $ \sqrt{x} $ $x$ 的平方根
sin(x) $\sin x$ $x$ 的正弦
cos(x) $\cos x$ $x$ 的余弦
tan(x) $\tan x$ $x$ 的正切
asin(x) $\arcsin x$ $x$ 的反正弦
acos(x) $\arccos x$ $x$ 的反余弦
atan(x) $\arctan x$ $x$ 的反正切
log(x) $\ln x$ $x$ 的自然对数（e 为底）
log10(x) $\log_{10}{x}$ $x$ 的常用对数（10 为底）
exp(x) $e^x$ e 的 $x$ 次方
ceil(x) 最小的 >=$x$ 的整数
floor(x) 最大的 <=$x$ 的整数

random 库

函数 含义
random() 返回在范围[0,1)中均匀分布的伪随机值
randrange(p1, p2) 返回在范围[p1, p2)的伪随机整数

序列

序列操作（字符串和列表）

操作符 含义
+ 返回序列的连接。序列必须是相同类型
* 返回序列与自身连接 n 次。n 必须是整数
[] where n < 0 返回右边 n 第 n 个数据项（最右边算 1）。n 必须是整数
len() 返回序列的长度
[:] 返回从 start 直到（但不包括）end 的子序列
for in 迭代序列中的数据项
in 成员检查

字符串方法

函数 含义
s.capitalize() 只有第一个字符大写的 s 的副本
s.center(width) 在给定宽度的字段中居中的 s 的副本
s.count(sub) 计算 s 中的 sub 的出现次数
s.find(sub) 找到 sub 出现在 s 中的第一个位置
s.join(list) 将列表连接到字符串中，使用 s 作为分隔符
s.ljust(width) 类似 center，但 s 是左对齐
s.lower() 所有字符小写的 s 的副本
s.lstrip() 删除前导空格的副本
s.replace(oldsub,newsub) 使用 newsub 替换 s 中的所有出现的 oldsub
s.rfind(sub) 类似 find，但近观回最右边的位置
s.rjust(width) 类似 center，但 s 是右对齐
s.rstrip() 删除尾部空格的 s 的副本
s.split() 将 s 分割成子字符串列表
s.title() s 的每个单词的第一个字符大写的副本
s.upper() 所有字符都转换为大写的 s 的副本

类型转换

函数 含义
float() 将 expr 转换为浮点值
int() 将 expr 转换为整数值
str() 返回 expr 的字符串表示形式
eval() 将字符串作为表达式求值

字符串格式化

表达式语法

.format(, , , …)

格式说明

{}

{:} ：value 数据长度不够 width，后面补空格

{:.}：只获取 value 数据的 precision 长度

{:.f }：保留 value 数据小数点前 places 位

  • 最后的形式是针对固定的小数位数。
  • 宽度为 0 表示不论需要多少空间。
  • 前导 0 的宽度表示必要时的填充为 0（默认为空格）。
  • 宽度前面可以有<表示左对齐，>用表示右对齐，或^表示中心对齐。

列表方法

方法 含义
.append(x) 将元素 x 添加到列表末尾
.sort() 对列表排序。关键字参数 key、reverse
.reverse() 反转列表
.index(x) 返回 x 首次出现的索引
.insert(i, x) 在列表的索引 i 处插入 x
.count(x) 返回 x 在列表中出现的次数
.remove(x) 删除列表中首次出现的 x
.pop(i) 删除列表中第 i 个元素并返回它的值

字典方法

字典字面量：{: , : , …}

方法 含义
in 如果字典包含指定的 key，就返回 True，否则返回 False
.keys() 返回键的序列
.values() 返回值的序列
.items() 返回元组（key,value）的序列，表示键值对
.get(,) 如果字典包含键 key 就返回它的值，否则返回默认值 defaul
del [] 删除指定的条目
.clear() 删除所有条目
for in : 循环遍历所有键

布尔值

字面量：True，False

操作符：and，or，not

操作符 操作定义
x and y 如果 x 为 false，返回 x，否则返回 y
x or y 如果 x 为 true，返回 x，否则返回 y
not x 如果 x 为 false，返回 True，否则返回 False
