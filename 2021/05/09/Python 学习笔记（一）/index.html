Python 学习笔记（一）

发表于 2021-05-09 分类于 后端 ， Python

变量和简单数据类型

变量

Python 中变量无需声明类型，会根据赋值的内容自动推导类型

变量命名注意事项：

  • 变量名只能包含字母、数字、下划线。下划线不能作为开头
  • 不能包含空格
  • 不使用 Python 关键字和函数名为变量名
  • 变量名应 “短小且精悍”（描述性好）

字符串

使用引号（单引号、双引号）括起的一系列字符

大小写相关函数：

函数名 作用
title 每个单词首字母大写
upper 全部大写
lower 全部小写

删除空格：

函数名 作用
rstrip 删除末尾空格
lstrip 删除开头空格
strip 删除两端空格

使用字符串注意事项：

  • 撇号 ' 要位于双引号 " " 之间，在单引号 ' ' 之间会报错
  • \n 表示换行，\t 表示制表符
  • 双引号之间可以使用单引号，在单引号之间可以用双引号

在 Python 2 中 print 语句可以无需括号包装内容，但在 Python 3 中必须需要括号

数字

Python 中将数字分为 整数 int 和 浮点数 float

Python 将带小数点的数字都称为浮点数

基本运算符：加（+）减（-）乘（*）除（/）

几次方运算：**

求模运算符：%

扩展运算：//（除取整）、+= 、-=、*=、/=、//=

数字运算注意事项：

  • Python 不支持自增减符 ++、- -
  • //（除取整）不是四舍五入，而是将小数部分直接删除
  • 在字符串中使用值为数字的变量时，需使用 str() 函数进行类型转换

Python 2 整数除法与 Python 3 的 // 结果相同，浮点数除法时才会保留小数

布尔值

在 Python 中使用 True 和 False 表示是否，与其他编程语言不同，它们的开头是大写字母

注释

在 Python中，注释用井号（#）标识

注释使用注意事项：

  • 编写注释的主要目的是阐述代码要做什么，以及是如何做的
  • 在代码中编写清晰、简洁的注释
