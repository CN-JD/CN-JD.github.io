Python 学习笔记（二）

发表于 2021-05-09 分类于 后端 ， Python

列表

列表是什么

列表由一系列按特定顺序排列的元素组成，类似 C 的数组

用方括号 [ ] 来表示列表，并用逗号来分隔其中的元素

创建空列表 motorcycles = []

列表索引从 0 开始，-1 表示列表最后一个元素

修改元素

要修改列表元素，可指定列表名和要修改的元素的索引，再指定该元素的新值

添加元素

  1. 在列表末尾添加元素 append()
  2. 在列表中插入元素 insert()，需指定新元素的索引和值

删除元素

  1. 使用 del 语句删除元素，无返回值

  2. 使用方法 pop() 删除元素，返回删除的元素，可指定删除元素的索引

  3. 根据值删除元素 remove()，无返回值

    如果列表有多个重复的元素，remove() 只能删除一个元素，需要循环才能删除所有

列表排序

永久性说明改变了列表本身

函数名 作用
sort 永久性排序
sorted 临时排序
reverse 反转列表
len 列表长度

向函数 sorted() 传递参数 reverse=True，按与字母顺序相反的顺序显示列表

操作列表

遍历列表

使用 Python中 的 for 循环对列表中的每个元素都执行相同的操作

Python 不使用大括号而是使用冒号 : 和缩进表示代码块

创建数值列表

  • Python 函数 range() 可以生成一系列的数字

    从指定的第一个值开始数，并在到达指定的第二个值后停止

  • 使用函数 list() 将 range() 的结果直接转换为数字列表

    需要反向生成一系列数字时必须使用步长

  • 使用函数 range() 时，可以指定第三个参数的值即步长，默认为 1

统计数字列表

函数名 作用
min 求最小值
max 求最大值
sum 求总和

切片

使用 : 处理列表的部分元素的操作

  • 指定开始索引和结束索引

  • 不指定开始索引，表示从开头开始

  • 不指定结束索引，表示到列表末尾

  • 不指定开始和结束索引，一般用于复制列表

遍历列表的部分元素，可在 for 循环中使用切片

复制列表

要正确获取一个列表的副本，就必须使用切片进行操作

这种语法实际上是让 Python 将新变量 friend_foods 关联到包含在 my_foods 中的列表，因此这两个变量都指向同一个列表

使用切片获取副本 新变量名 = 列表[:]

元组

元组是什么

不可变的列表被称为元组，即元组在定义后，不能在添加修改或删除其中的元素

用元括号 ( ) 来表示元组，并用逗号来分隔其中的元素

可以修改元组这个变量

遍历元组

使用 for 循环来遍历元组中的所有值
