字典

什么是字典

字典是一系列键–值对

每个键都与一个值相关联，使用键来访问与之相关联的值

字典用放在花括号 { } 中的一系列 键: 值 对表示

与键相关联的值可以是数字、字符串、列表乃至字典即任何对象

字典里的键名唯一，但值不唯一

创建空字典 student = {}

添加键值对

字典是一种动态结构，可随时在其中添加键–值对 字典名[ 新键 ] = 新值

修改字典的值

依次指定字典名、用方括号括起的键以及与该键相关联的新值 字典名[ 键 ] = 新值

删除键值对

使用 del 语句将相应的键—值对彻底删除 del 字典名[ 键 ]

遍历字典

函数 作用
items 返回 <键，值>
keys 返回 <键>
values 返回<值>

  • 遍历所有的键值对 for k, v in 字典名.items()

  • 遍历所有键 for k in 字典名.keys()

  • 按顺序遍历所有键 for k in sorted(字典名.keys())

  • 遍历所有值 for v in 字典名.values()

嵌套

将一系列字典存储在列表中，或将列表作为值存储在字典中，这称为嵌套

字典列表

列表中的元素为字典

字典存储列表

字典的某个键的值为列表

字典存储字典

字典某个键的值为字典
