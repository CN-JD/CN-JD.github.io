用户输入

input() 获取字符串输入

使用函数 input() 接收用户的输入

函数 input() 接受一个参数：即要向用户显示的提示或说明

函数 input() 将用户输入解读为字符串

Python 2.7 中利用 raw_input() 用来提示用户输入

int() 转换为数值

使用函数 int() 可将一个字符串格式的数值转换为整型的数值

While 循环

for 循环用于针对集合中的每个元素都一个代码块，而 while 循环不断地运行，直到指定的条
件不满足为止

break 退出循环

在任何Python循环中都可使用break语句。例如，可使用break语句来退出遍历列表或字典 的for循环

continue

返回到循环开头，并根据条件测试结果决定是否继续执行循环
