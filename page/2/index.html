简介

批处理定义：批处理文件是将一系列命令按一定的顺序集合为一个可执行的文本文件，其扩展名为BAT 或者 CMD 。这些命令统称批处理命令。

其实，我们在 CMD 中输入的例如 cd、ping 命令都是批处理命令。

在 windows 平台下编写一个批处理文件，有利与我们处理一些繁琐且重复的活动，例如本文介绍的利用批处理命令编写一个用于管理 Hexo 博客的批处理脚本。
