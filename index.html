Python 学习笔记（六）

发表于 2021-05-10 分类于 后端 ， Python

函数

定义函数

函数是带名字的代码块，用于完成具体的工作

传递实参

当函数定义中包含多个形参，在函数调用时也可能要包含多个实参

位置实参：基于实参的顺序关联形参

调用中实参的顺序与函数定义中形参的顺序要一致

关键字实参：实参传递键值对

必须准确地指定函数定义中的形参名

默认值：在函数定义时，形参可以指定默认值

多个形参中有一个形参指定默认值，其后面的形参也必须指定默认值

返回值

Python 使用 return 语句将值返回到调用函数的代码行

传递任意数量的实参

利用 *形参 可以传递任意个实参

  • 结合使用位置实参和任意数量实参

  • 使用任意数量的关键字实参

将函数存储在模块

函数存储在被称为模块的独立文件中，再将模块的函数导 入到主程序中调用

  • 导入整个模块

    使用 import 模块名 导入模块，可以调用模块中所有函数

    pizza.py

    making_pizzas.py

  • 导入特定函数

    使用 from 模块名 import 函数名, 函数名 导入函数，可以调用模块部分特定函数

    making_pizzas2.py

  • 使用 as 给函数指定别名

    如果要导入的函数名与程序中现有函数名的冲突，可利用 as 指定函数的别名

    making_pizzas3.py

  • 导入模块中的所有函数

    使用 from 模块名 import * 复制模块所有函数到程序中，直接使用，无需在使用 模块名.函数名 调用

    making_pizzas4.py

函数编写指北

  • 函数名且只使用小写字母和下划线

  • 函数定义后面应包含简要地阐述其功能的注释

  • 给形参指定默认值和使用关键字实参时，等号两边不要有空格

  • 代码行的长度不要超过79字
